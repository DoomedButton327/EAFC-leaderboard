# Mettlestate × EA FC Mobile — Public Leaderboard

Live standings and fixtures for the Mettlestate EA FC Mobile League.

## 🌐 Live Site

Visit the public leaderboard at your GitHub Pages URL after enabling Pages in repo settings.

## 📂 How it works

- `index.html` — read-only viewer (Standings + Fixtures tabs)
- `data/league-data.json` — league data pushed here from the admin panel

Data is pushed automatically from the main admin site whenever an admin clicks  
**"Push to Public Leaderboard"** in the Admin panel.

> **This repo is read-only for players.** No editing happens here.  
> Admin management happens in the private admin repo.

## ⚙️ Setup

1. Create this repo on GitHub (can be public)
2. Go to **Settings → Pages → Source: GitHub Actions**
3. In the admin panel of your main site, enter this repo's details under **Public Leaderboard Sync**
4. Click **Push to Public Leaderboard** to publish standings
